# ブランドガイドライン

最終更新日: 2025-10-30

本ガイドラインは、アプリ内のUI、広告表示、外部ブランドの扱いに関する基準を定め、法令およびプラットフォーム規約に準拠した運用を支援するものです。

## 1. コアブランド要素

- **サービス名**: カフェ・レストラン可視化アプリ（仮称）
- **トーン&マナー**: 落ち着いた雰囲気、信頼性、ユーザーが安心して店舗を探せる体験。
- **カラーパレット**: 高コントラスト比を確保し、色覚多様性に配慮した配色（詳細はデザインシステムに準拠）。
- **ロゴ・アイコン**: 開発段階ではシンプルなタイポグラフィを使用し、正式版でロゴを定義する。必ずSVG/PNGの公式素材を使用する。

## 2. 広告表示ルール

1. **ラベル表示**
   - リスト内広告、地図上のプロモーションピン、バナー広告には必ず「広告」または「PR」ラベルを表示。
   - ラベルは視認性の高い位置に配置し、非広告コンテンツとの差別化を明確にする。
2. **広告主情報**
   - 広告枠の下部または詳細ポップアップに広告主名称や問い合わせ先（URL等）を併記。
   - アフィリエイトリンクの場合、「リンク先で購入すると当方が収益を得ることがあります」などの説明を追加。
3. **表現ルール**
   - 景品表示法、薬機法、健康増進法等に抵触する誇大表示・効果効能表示を避ける。
   - クーポンやキャンペーンには適用条件・有効期限を明記。
4. **ターゲティング**
   - 行動履歴に応じた広告を表示する場合、プライバシーポリシーで利用目的とオプトアウト方法を明示。

## 3. コンテンツ掲載ガイドライン

- 飲食店の写真や説明文は、引用元や広告主提供素材の利用許諾を確認し、出典を表示。
- ユーザーが投稿したお気に入りコメント等を広告に転用する場合は事前同意を取得。
- 口コミ引用は Google Places の制限（最大5件）を守り、レビュアー名・投稿日を併記。

## 4. プラットフォーム別ガイドライン抜粋

| プラットフォーム                      | 主な遵守事項                                                                                                  |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Google Maps Platform                  | © Google の帰属表示。地図タイル・POIデータのキャッシュ禁止。規約: <https://developers.google.com/maps/terms> |
| Google Places                         | レビューを加工・改変しない。保存はキャッシュ扱いに留め、最新情報取得を基本とする。                            |
| Instagram / Meta                      | Graph API利用時は Meta Platform Policy を遵守し、スクリーンショット公開時はユーザー名と出典を表記。           |
| 広告ネットワーク (例: Google AdSense) | 配置ポリシー、アダルト/危険コンテンツ禁止、自己クリック禁止を遵守。                                           |

## 5. UIコンポーネント要件

- **広告カード**: 背景色または枠線で通常カードと差別化し、左上に「広告」ラベル、右下に広告主ロゴまたは名称。
- **地図ピン**: プロモーションピンは通常ピンと異なる色＋「広告」ツールチップを表示。
- **お気に入りリスト**: 広告枠を設ける場合はリスト最上部/最下部に固定し、ユーザー操作を妨げない配置にする。
- **モーダル/詳細画面**: 広告主情報（名称、URL、問い合わせ先）を表示し、必要に応じて「提供：〇〇」表記。

## 6. データ・ロゴ利用

- 各飲食店の公式ロゴ・写真を掲載する場合は、公式サイトの利用規約または個別許諾を確認。
- SNS投稿の埋め込みはAPIが提供する公式ウィジェットを活用するか、引用要件（ユーザー名、リンク）を満たす。
- 無断転載が疑われる素材を発見した場合は速やかに削除し、出典確認を実施。

## 7. 品質管理フロー

1. 広告素材・PR記事は事前にチェックリストで表現と帰属表示を確認。
2. UI更新時はアクセシビリティ（コントラスト、フォントサイズ、キーボード操作）を確認。
3. 半期ごとにプラットフォーム規約や法令の更新を調査し、ガイドラインをアップデート。

## 8. 改定

本ガイドラインは必要に応じて改定します。主要な変更はリポジトリのCHANGELOGまたはアプリ内通知で周知します。
